# FileCollector v2.1

FileCollector — WinForms-приложение для Windows, которое автоматизирует поиск и сборку файлов по списку из Excel. Приложение рекурсивно ищет файлы в заданных папках, показывает предпросмотр и копирует выбранные файлы с отчётом о результатах.

## Возможности
- Чтение списка имён файлов из Excel (столбец B первой страницы)
- Рекурсивный поиск по множеству расширений
- Добавление пользовательских расширений
- Предпросмотр найденных файлов перед копированием
- Выборочное копирование и отчётность

## Требования
- Windows 10/11 x64
- .NET SDK 10.0 (только для сборки из исходников)

## Сборка и запуск

```powershell
dotnet restore
dotnet build
dotnet run --configuration Debug
```

## Release-сборка (актуальная версия)

```powershell
dotnet build -c Release
```

Исполняемый файл будет здесь:
`bin/Release/net10.0-windows/win-x64/FileCollector.exe`

## Публикация одиночного исполняемого файла

```powershell
dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true
```

Результат:
`bin/Release/net10.0-windows/win-x64/publish/FileCollector.exe`

## Быстрый старт
1. Подготовьте Excel файл: имена файлов в столбце B первой страницы.
2. Откройте приложение и выберите Excel файл.
3. Добавьте папки для поиска.
4. Отметьте нужные расширения и папку для сбора.
5. Запустите поиск и копирование.

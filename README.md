# FileCollector v2.0 — Интеллектуальный сборщик файлов

## Описание

**FileCollector** — это мощное WinForms-приложение для Windows, которое автоматизирует поиск и сборку файлов из разных расширений и локаций. Приложение читает список имён файлов из Excel, рекурсивно ищет их во множестве папок с поддержкой параллельного поиска, показывает интерактивный предпросмотр и копирует выбранные файлы с подробным отчётом о результатах.

## Ключевые особенности

✨ **Основной функционал:**
- 📝 Чтение списка имён файлов из Excel (столбец B первой страницы)
- 🔍 Рекурсивный поиск файлов по множеству расширений (более 30 форматов по умолчанию)
- 📂 Поддержка добавления пользовательских расширений
- 👁️ Интерактивный предпросмотр найденных файлов перед копированием
- ✅ Выборочное копирование (галочки в таблице предпросмотра)
- 💾 Массовое копирование с отчётностью о результатах

⚡ **Производительность:**
- Параллельный обход подпапок (`Parallel.ForEach`) для ускорения на многоядерных машинах
- Кэширование файловой системы на 5 минут для оптимизации многократных поисков
- Использование `EnumerateFiles`/`EnumerateDirectories` вместо `GetFiles`/`GetDirectories` для меньших аллокаций памяти
- Потокобезопасное обновление результатов поиска с использованием `lock`

🛡️ **Надёжность:**
- Обработка заблокированных файлов (попытки повтора, пропуск при невозможности)
- Игнорирование системных и скрытых файлов (флаг `skipSystemHidden`)
- Подробное логирование всех операций
- Генерация отчётов о ненайденных файлах
- Поддержка отмены операции в процессе выполнения

🎨 **Пользовательский интерфейс:**
- Drag-and-drop для добавления Excel файлов и папок поиска
- Горячие клавиши: `Ctrl+O` (открыть Excel), `Ctrl+P` (предпросмотр), `Ctrl+S` (запуск), `Esc` (отмена)
- Прогресс-бар с отчётом о ходе операции
- Контекстные меню для работы с файлами
- Подробное логирование процесса поиска и копирования

## Установка и запуск

### Вариант 1: Запуск исполняемого файла (рекомендуется)

1. Загрузите `FileCollector.exe` из папки `bin/Release/net10.0-windows/win-x64/publish/`
2. Запустите файл — не требует установки .NET Runtime

### Вариант 2: Локальная сборка и запуск

```powershell
dotnet build
dotnet run --configuration Debug
```

### Вариант 3: Публикация одиночного исполняемого файла

```powershell
dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true
```

Артефакты будут в `bin/Release/net10.0-windows/win-x64/publish/FileCollector.exe`

## Быстрый старт

1. **Подготовьте Excel файл:**
   - Поместите имена файлов в столбец B первой страницы
   - Имена могут быть с расширением или без

2. **Запустите приложение:**
   - Откройте FileCollector.exe

3. **Выберите параметры поиска:**
   - Нажмите "Выбрать Excel" и откройте ваш файл
   - Добавьте папки для поиска (можно drag-and-drop)
   - Отметьте нужные расширения (по умолчанию `.pdf` и `.dxf`)
   - Выберите папку для сбора файлов

4. **Предпросмотр:**
   - Нажмите "Предпросмотр" чтобы увидеть найденные файлы
   - Снимите галочки с ненужных файлов

5. **Копирование:**
   - Нажмите "Запустить копирование"
   - Приложение скопирует выбранные файлы и создаст отчёт

## Оптимизация производительности

**v2.0 Улучшения:**
- ✅ Параллельный обход подпапок (Parallel.ForEach с MaxDegreeOfParallelism = ProcessorCount)
- ✅ Использование EnumerateFiles вместо GetFiles для ленивых вычислений
- ✅ Кэширование результатов сканирования с автоматической очисткой
- ✅ Потокобезопасные обновления latestFile с использованием lock

Благодаря этим оптимизациям на многоядерных машинах поиск может быть в 2-4 раза быстрее, особенно при работе с глубокими иерархиями папок.
